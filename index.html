隐私政策 / Privacy Policy

本应用非常重视用户的隐私与个人信息保护。本隐私政策旨在说明我们如何收集、使用和保护您的信息。请您在使用本应用前仔细阅读本政策。 This app highly values user privacy and personal information protection. This privacy policy explains how we collect, use, and protect your information. Please read it carefully before using the app.

一、信息收集与使用 / 1. Information Collection and Use

在您使用本应用过程中，我们可能会收集以下与运维管理相关的信息： While using this app, we may collect the following information related to maintenance management:

  • 账号信息（如用户名、登录凭证，用于身份验证和用户身份识别）
    Account information (such as username and login credentials, used for authentication and user identification)
  • 设备运维数据（工单记录、耗材更换情况、故障处理信息等）
    Device maintenance data (work order records, consumable replacement, fault handling information, etc.)
  • 设备运行状态及告警信息，用于实时监控和运维管理
    Device operational status and alert information for real-time monitoring and maintenance management

以上信息仅用于实现应用的核心功能，如设备运维管理、工单处理、数据展示和故障预警，不会用于与运维无关的用途。
The above information is only used for core functions such as maintenance management, work order processing, data display, and fault alerts, and will not be used for purposes unrelated to maintenance.

二、信息存储与安全 / 2. Data Storage and Security

我们采取合理的技术和管理措施保护您的信息安全，防止数据被未经授权访问、披露、篡改或丢失。
We adopt reasonable technical and administrative measures to protect your information from unauthorized access, disclosure, alteration, or loss.

三、信息共享与披露 / 3. Information Sharing and Disclosure

未经用户明确同意，我们不会向任何第三方披露、出售或共享您的个人信息，法律法规另有规定的除外。
We will not disclose, sell, or share your personal information with any third party without your explicit consent, except as required by law.

四、应用权限说明 / 4. App Permissions Explanation

为了实现应用的核心功能，本应用可能会请求以下设备权限，并严格按照功能使用：
To provide core functionality, this app may request the following device permissions and will strictly use them according to the function:

  • 网络权限 / Network Permission：用于与服务器同步数据、获取工单信息、更新设备状态及发送告警通知。
    示例 / Example: 应用通过网络获取最新的运维任务和工单记录。
    Used to sync data with the server, retrieve work order information, update device status, and send alerts. Example: The app retrieves the latest maintenance tasks and work order records via the network.
  • 相机权限 / Camera Permission：用于现场拍照记录运维情况。
    示例 / Example: 用户可拍摄设备故障照片或运维完成情况，便于生成工单报告。
    Used to take photos on-site for maintenance records. Example: Users can take photos of device faults or completed maintenance to generate work order reports.
  • 照片/存储权限 / Photo/Storage Permission：用于保存拍摄的图片或导出的工单报告。
    示例 / Example: 用户拍摄的运维现场照片会保存到应用或本地相册，以便随时查看和生成报告。
    Used to save captured photos or exported work order reports. Example: Maintenance photos taken by users are saved to the app or local album for later viewing and report generation.
  • 定位权限（可选） / Location Permission (Optional)：用于记录设备或运维现场的地理位置。
    示例 / Example: 在提交工单或拍照时记录当前位置，方便管理人员了解现场情况。
    Used to record the geographic location of devices or maintenance sites. Example: Records current location when submitting a work order or taking photos to help managers understand the site situation.
  • 通知权限（可选） / Notification Permission (Optional)：用于提醒用户新的工单、状态更新或告警信息。
    示例 / Example: 当设备状态异常或工单被分配时，应用通过通知提醒用户及时处理。
    Used to notify users of new work orders, status updates, or alerts. Example: The app notifies users when a device status is abnormal or a work order is assigned.

所有权限均在获得用户授权后才会使用，应用不会在未经允许的情况下访问相机、存储、位置信息或发送通知。
All permissions are only used after obtaining user authorization. The app will not access camera, storage, location, or send notifications without permission.
